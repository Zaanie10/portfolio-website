# Portfolio Website

A personal software engineering portfolio showcasing projects, technical skills, and professional experience.

**Live Site:** https://zaaniebowen.dev

---

## Overview

This portfolio website serves as a central hub for my software engineering work, highlighting selected projects, technical skills, and professional experience in a clean, responsive, and recruiter-friendly format.

Built using HTML, CSS, JavaScript, and Bootstrap, the website began with the Start Bootstrap Personal template and was extensively customized to reflect my own design preferences, content, and professional goals.

---

## Built With

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Responsive Web Design

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Professional landing page with technical skills overview
- Dedicated projects page showcasing development work
- Online resume page with downloadable PDF resume
- Direct links to GitHub, LinkedIn, and contact information
- Customized styling, layout, navigation, and content

---

## Site Structure

### Home

Introduces my background, technical skills, and career interests as a software engineer.

### Projects

Showcases selected software development projects, technologies used, and project details.

### Resume

Provides an online version of my resume along with a downloadable PDF version.
